# P800 知识

## 评估时检查

- Python 与实际 import 路径；
- torch、引擎、Kunlun 插件和本地算子版本；
- P800 可见性与设备数；
- 动态库、磁盘和 `/dev/shm`；
- TP/EP 通信与设备映射；
- 当前目标仓文档声明的版本兼容性。

## 实现时检查

同名算子不代表语义兼容。必须比较数学语义、shape、dtype、layout、mask/routing、prefill/decode 和并行行为。

禁止通过 CPU/CUDA fallback、吞异常或占位输出伪装 P800 支持。

具体环境变量和工具命令必须以当前容器和目标仓为准，不在通用知识中硬编码。
